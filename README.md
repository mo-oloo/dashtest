# dashtest

## Overview
This is a Dash App for an interactive dashboard to solve first order linear ODEs with initial values.

## Prerequisites
- Python 3.11

## Installation
Instructions to install dependencies:
```bash
pip3 install -r requirements.txt
```

## Running Locally
To run this app locally:
```bash
python3 dashtest.py
```
